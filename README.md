# Modern Web Calculator

A sleek, modern, and responsive web-based calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Sleek glassmorphism design with a premium feel
- Responsive layout
- Smooth micro-animations for button interactions

## How to Run
Simply open `index.html` in any modern web browser to start using the calculator. No build steps or server required!
